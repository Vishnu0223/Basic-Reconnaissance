# Commands Used

## Ping
```bash
ping scanme.nmap.org
```

## DNS Lookup
```bash
nslookup scanme.nmap.org
```

## WHOIS
```bash
whois scanme.nmap.org
```

## DIG
```bash
dig scanme.nmap.org
```

## Basic Nmap Scan
```bash
nmap scanme.nmap.org
```

## Service Detection
```bash
sudo nmap -sV scanme.nmap.org
```

## OS Detection
```bash
sudo nmap -O scanme.nmap.org
```

## Aggressive Scan
```bash
sudo nmap -A scanme.nmap.org
```

## Traceroute
```bash
traceroute scanme.nmap.org
```

## Technology Detection
```bash
whatweb https://scanme.nmap.org
```

## Banner Grabbing
```bash
nc scanme.nmap.org 80
```
