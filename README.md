# Basic Reconnaissance

## Overview

This project demonstrates the reconnaissance phase of ethical hacking using open-source intelligence (OSINT) and network enumeration techniques. The objective is to gather information about a publicly authorized target using industry-standard tools.

## Objectives

- Passive Reconnaissance
- Active Reconnaissance
- DNS Enumeration
- Network Scanning
- Service Enumeration
- Technology Detection

## Tools

- Kali Linux
- Nmap
- Whois
- Dig
- Nslookup
- WhatWeb
- Netcat
- Traceroute

## Skills

- Linux
- Networking
- Ethical Hacking
- OSINT
- Information Gathering
